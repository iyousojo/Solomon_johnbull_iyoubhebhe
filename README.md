# My Developer Portfolio 🚀

[![React](https://img.shields.io/badge/React-19-blue?style=for-the-badge&logo=react)](https://react.dev/)
[![Vite](https://img.shields.io/badge/Vite-5-orange?style=for-the-badge&logo=vite)](https://vitejs.dev/)
[![TailwindCSS](https://img.shields.io/badge/TailwindCSS-4-38bdf8?style=for-the-badge&logo=tailwindcss)](https://tailwindcss.com/)

Modern, responsive personal portfolio website showcasing full-stack development projects. Built with **React + Vite + TailwindCSS**, featuring smooth **Framer Motion** animations, **BentoGrid** project layout, **EmailJS** contact form, and **WIT-powered AI Chatbot**.

## ✨ Features
- **Responsive Design**: Mobile-first, TailwindCSS-powered layouts
- **Smooth Animations**: Framer Motion + ScrollReveal effects
- **Project Showcase**: 5 featured full-stack apps with live demos, GitHub, APIs
- **Interactive Contact**: EmailJS integration for direct messages
- **AI Chatbot**: WIT service for real-time assistance
- **Performance Optimized**: Vite HMR, lazy loading, code splitting

## 🚀 Quick Start
```bash
# Clone & Install
git clone <your-repo-url>
cd my-portfolio
npm install

# Development
npm run dev

# Build & Preview
npm run build
npm run preview
```

Open [http://localhost:5173](http://localhost:5173) to view in browser.
